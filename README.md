## 👤 Sobre mí

Soy analista de datos junior de 27 años, con base en Argentina. Tengo **2 años de experiencia** como analista GTR en un call center, donde no me limité a reportar datos: identifiqué cuellos de botella operativos e implementé **automatizaciones con Python y macros de Excel** que eliminaron tareas manuales y aceleraron la entrega de reportes diarios.

Me formé de manera autodidacta y estructurada a través de **Platzi**, completando la ruta de análisis de datos y la ruta completa de Excel. Estoy construyendo un camino sólido hacia roles donde los datos realmente generen impacto.
